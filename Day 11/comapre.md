# Full-Doc vs RAG (Simple Explanation)

## Full-Document Approach

* You send the **whole file** to AI every time you ask a question
* Example: sending entire book again and again
* ❌ Uses a lot of tokens → expensive
* ❌ Slow for big files

## RAG Approach

* You **split the file into small parts (chunks)**
* AI only reads **important parts** for the question
* Example: only reading relevant pages, not full book
* ✅ Uses fewer tokens → cheaper
* ✅ Faster and scalable

## Simple Difference

* Full-Doc = give whole book every time
* RAG = give only useful pages

## Conclusion

RAG is better because it is **cheaper, faster, and smarter for big data**.
