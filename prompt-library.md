# Personal Prompt Library

This is my reusable collection of AI prompts for development, debugging, and productivity.

# 1. Code Generator Prompt
Generate clean, production-ready code for [technology]. Include proper structure, comments, and best practices.

# 2. Bug Fix Prompt
Analyze the given code and:
1. Identify the error
2. Explain the root cause
3. Provide corrected code

# 3. API Builder Prompt
Create a REST API using Node.js and Express for [feature]. Include CRUD operations and proper error handling.

# 4. UI Component Prompt
Create a responsive UI component using React and Tailwind CSS for [component name].

# 5. Project Architecture Prompt
Design a scalable architecture for a [project type] application including frontend, backend, and database structure.

# 6. SQL Query Prompt
Write optimized SQL queries for [task] and explain the logic step by step.

# 7. Resume Improvement Prompt
Improve this resume content to make it more professional and ATS-friendly.

# 8. Learning Explanation Prompt
Explain [topic] in simple terms with real-world examples and beginner-friendly language.

# 9. Code Refactoring Prompt
Refactor the given code to improve readability, performance, and maintainability without changing functionality.

# 10. Project Idea Generator Prompt
Suggest 5 unique full-stack project ideas for a beginner MERN stack developer with features and tech stack.
